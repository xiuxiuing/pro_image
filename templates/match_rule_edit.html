基本信息

全局默认（七维）

此处勾选＝对全表所有一级类目使用同一套标准（除非在下一栏为某个一级类目单独写了例外）。取消勾选＝该条目不参与过滤。

按「美团一级类目」的例外（可选，高级）

大部分用户可整栏留空。仅当不同行业需要不同松紧度时，用 JSON 为某个一级类目覆盖上面的默认。键必须与 Excel 里「美团一级类目」文字完全一致（含括弧、空格需一致）。

技术说明与字段名（与保存到数据库的 JSON 一致）

写成一个对象：外层键＝「美团一级类目」名称；内层可只写要改的维度。维度短键为 cat3, net, sell, pack, color, size, model；是否启用为 "en": true/false。会与「全局默认」深度合并，不必重复写整份规则。

示例：仅把「食品饮料」下的净含量校验关掉、其余继承默认——{"食品饮料":{"net":{"en":false}}}。售卖数量在默认里全表启用、仅「生鲜」要关掉——{"生鲜":{"sell":{"en":false}}}。
